# Inline Booking Automation Test

This project demonstrates an automation test case using **Playwright with C#**.

## Test Steps
1. Open Inline booking page
2. Select:
   - Party size: 3 adults
   - Dining date: Feb 11
   - Time: 12:00
3. Click "Complete booking"
4. Take a full-page snapshot

## Tech Stack
- .NET 8
- Microsoft Playwright
- C#

## How to Run
```bash
dotnet restore
dotnet build
dotnet run
